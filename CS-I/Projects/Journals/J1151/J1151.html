2.1: Why is the first jump conditional? What determines whether the jump will be executed or not?
The first jump is conditional because it should only jump to run certain code if the right condition is met. This jump is determined on what the value in the rax register is, specifically if the 2nd number compared is less than or equal to it.

2.2: What is the purpose of the jump intruction after the consequent?
The jump after consequent is to skip the code that should only be run if the previous output was alternative

2.3: Why is there no need for a corresponding jump instruction after the alternative?
There is no need for a jump because the code will continue to execute sequentially, and the consequent processes are before the alternative processes in the memory
